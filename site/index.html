Libscore

Libscore scans the top million sites to detect JavaScript library usage.

It provides open source developers the data they need to measure their impact.

It detects modules loaded via RequireJS, jQuery plugins, window variables produced by non-jQuery libraries, and external scripts.

Libscore was created by Julian Shapiro. Thomas Davis built the API.

It's sponsored by Stripe and Digital Ocean.
