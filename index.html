How do I use LibScore?

In the search box above, enter the variable that a JavaScript library exposes itself under, e.g. jQuery, Modernizr, $.ui, or $.fn.fancybox.

You can also search for sites containing an external script by prefixing a query with script:, e.g. script:stats.wp.com or script:use.typekit.net.

Alternatively, you can retrieve all libraries used on a site by querying for its domain name, e.g. stripe.com or digitalocean.com.
